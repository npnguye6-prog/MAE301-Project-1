# AeroPredict MVP (Phase 3)

This MVP predicts Remaining Useful Life (RUL) of turbofan engines using sensor data.  
It outputs predicted RUL, uncertainty bounds, and model evaluation metrics.

## How to run

```bash
python aeropredict_mvp.py
```
## Output
- Predicted Remaining Useful Life (RUL)
- Confidence interval (uncertainty bounds)
- Model evaluation metric (MAE)

## Example Output

```text
Predicted RUL: 47 cycles
Confidence interval: [40, 55]
MAE: 6.2
```
